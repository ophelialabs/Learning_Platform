## Curriculum Overview

### Core Courses (Weeks 1-20)

#### Foundation: Building Blocks (Weeks 1-6)
1. **Program Induction & Fundamentals** (2 weeks)
   - Welcome & orientation
   - Basic statistics
   - Environment setup
   - *Modules*: Program Induction, Statistics Basics, Environment Setup

2. **Programming Refresher** (3 weeks)
   - Python fundamentals
   - Data structures
   - Functions & modules
   - *Modules*: Python Basics, Data Structures, Functions

3. **Database Management using SQL** (4 weeks)
   - SQL fundamentals
   - Advanced queries
   - Database design
   - *Modules*: SQL Fundamentals, Advanced Queries, Database Design

#### Data Science Essentials: Tools & Techniques (Weeks 7-14)
4. **Python for Data Science (IBM)** (4 weeks)
   - Jupyter & Pandas
   - Data cleaning
   - Visualization
   - *Modules*: Jupyter Pandas, Data Cleaning, Visualization

5. **Applied Data Science with Python** (5 weeks)
   - NumPy & Linear Algebra
   - Statistics for DS
   - Feature engineering
   - *Modules*: NumPy/Linear Algebra, Statistics, Feature Engineering

6. **Machine Learning** (6 weeks)
   - Supervised learning
   - Unsupervised learning
   - Ensemble methods
   - *Modules*: Supervised Learning, Unsupervised Learning, Ensemble Methods

#### Advanced Topics: Next-Generation Technology (Weeks 15-20)
7. **Deep Learning Specialization** (6 weeks)
   - Neural networks
   - CNNs
   - RNNs
   - *Modules*: NN Fundamentals, CNNs, RNNs

8. **Essentials of Generative AI & Prompt Engineering** (4 weeks)
   - GenAI fundamentals
   - Prompt engineering
   - ChatGPT & LLMs
   - *Modules*: GenAI Basics, Prompt Engineering, ChatGPT

9. **Generative AI for Data Professionals** (5 weeks)
   - GenAI for data generation
   - GenAI in data pipeline
   - Industry applications
   - *Modules*: Data Generation, Data Pipeline, Applications

10. **Advanced GenAI Skills for Data Scientists (IBM)** (5 weeks)
    - GANs & VAEs
    - Data prep with GenAI
    - Predictive modeling
    - *Modules*: GANs/VAEs, Data Preparation, Predictive Modeling

### Capstone & Specialization (Weeks 21-28)

11. **Data Science Capstone Project** (8 weeks)
    - End-to-end project
    - Real-world problem solving
    - Industry mentorship
    - *Modules*: Planning & EDA, Model Development, Visualization & Presentation

12. **Electives & Specializations** (Choose 1-3)
    - **Data Visualization using Power BI** (4 weeks)
    - **Advanced Deep Learning & Computer Vision** (5 weeks)
    - **Natural Language Processing** (5 weeks)
    - **R Programming for Data Science** (4 weeks)
    - **Data Ethics** (3 weeks)
    - **Industry Masterclass by IBM** (3-4 weeks)

## Getting Started

### Step 1: Set Up Your Environment
```bash
# Create directory
mkdir data-science-learning
cd data-science-learning

# Create virtual environment
python -m venv ds_env
source ds_env/bin/activate

# Install dependencies (see RESOURCES.md)
pip install -r requirements.txt

# Launch Jupyter
jupyter lab
```

### Step 2: Access Learning Materials
1. Open **[learning-platform.html](learning-platform.html)** in browser
2. Read **[LEARNING_GUIDE.md](LEARNING_GUIDE.md)**
3. Browse **[course/](course/)** directories

### Step 3: Start Learning
- Begin with Course 1: Program Induction
- Follow the structured modules
- Complete hands-on exercises
- Build projects
- Track your progress
